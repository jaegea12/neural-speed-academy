# Changelog

## v1.0.0 — Release (2025-06-13)

First public release of Neural Speed Academy.

### Exercises (15)

**Perception (5):**
- Flash Perception — rapid digit/letter recognition (#1 precursor)
- Peripheral Flash — peripheral vision identification (#3)
- Schulte Grid — visual field expansion with numbered grids
- Multiple Object Tracking (MOT) — divided attention and visual tracking (#5)
- Eye Priming — guided eye warmup (saccade, pursuit, convergence)

**Cognition (5):**
- Reaction Time — simple, choice, and go/no-go modes (#8)
- Rapid Decision Grid — rule-based grid search with multiple modes (#4)
- Sequence Memory — progressive sequence recall (#1)
- Split Attention — dual-task word + shape identification (#7)
- Slide Processing — timed information extraction with comprehension questions (#9, #13)

**Reading (5):**
- Guided Pacer — highlight-based reading with adjustable WPM and page width (#48)
- RSVP Reader — rapid serial visual presentation
- Chunked Reading — phrase-level reading with configurable chunk size
- Spaced Repetition — flashcard system with deck management (#6)
- Recall Training — post-reading comprehension assessment (#2)

### Training Paths

- 16 structured paths across 5 categories: Daily Routines, Speed Reading, Cognitive Performance, Visual Processing, Information Processing (#14)
- Custom path builder with exercise catalog

### Stats & Analytics (#10, #18–#21)

- XP system with levels and streak tracking
- Per-exercise trend charts
- Personal bests display
- Session history with timestamps, scores, and metadata
- Strengths/weaknesses analysis
- Training consistency heatmap (12-week view)
- Data export in CSV and JSON formats (#40)

### Theme Engine & Accessibility (#29–#31, #33)

- 11 color profiles: Default Dark, Midnight, Warm Dark, Dark Red, Dark Blue, Light Blue, Solarized, High Contrast, Deuteranopia, Protanopia, Tritanopia
- Font scale slider (80%–150%)
- Keyboard shortcuts for all core actions (Ctrl+Enter, Space, Escape, F11)
- Focus indicators and screen reader hints

### Internationalization (#11)

- 6 languages: English, German, French, Spanish, Italian, Portuguese
- 523 translation keys per locale covering all UI strings, exercise guides, intro content, result details
- 9 native training texts per language
- Language selector with flag images
- Locale-aware text library

### UI & Polish (#15–#17, #25–#28, #38)

- Unified exercise config menu with two-panel layout, presets, and guide panels
- 3-column exercise grid on dashboard
- Consistent navigation bar with back button, breadcrumb, and stop button
- Main menu with headline, tagline, level/XP display
- Slide creator for custom slide sets (#41)
- App icon and branding (#47)
- Audio feedback for correct/incorrect, timer warnings, completion (#32)

### Bug Fixes (#26, #34, #36, #37)

- Dashboard button text clipping in windowed mode
- Enter key not submitting on profile creation
- Split Attention always showing wrong result
- Sequence Memory mixed mode showing numbers in word selection row
